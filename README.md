# 💜 Clone do App Nubank
Um projeto desenvolvido com **React Native** e **TypeScript**, inspirado na interface do aplicativo **Nubank**.  
O objetivo é **recriar a tela principal** do app de forma educativa, explorando conceitos de UI, componentes reutilizáveis e animações.

- 💸 Serviços bancários (Área Pix, Pagar, Caixinha, etc)
- 💬 Carrossel de mensagens
- 💳 Cartão de crédito
- 💰 Empréstimo
- 🌟 Descubra mais
- 🔘 Botões flutuantes
